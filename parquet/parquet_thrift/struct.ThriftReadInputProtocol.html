pub(crate) struct ThriftReadInputProtocol<R: Read> {
    reader: R,
}
Expand description

A Thrift input protocol that wraps a Read object.

Note that this is only intended for use in reading Parquet page headers. This will panic if Thrift binary data is encountered because a slice of that data cannot be returned.
