Expand description

Helper to create CommandGetSqlInfo responses.

CommandGetSqlInfo are metadata requests used by a Flight SQL server to communicate supported capabilities to Flight SQL clients.

Servers constuct - usually static - SqlInfoData via the SqlInfoDataBuilder, and build responses using CommandGetSqlInfo::into_builder
