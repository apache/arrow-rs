Represents a SQL query. Used in the command member of FlightDescriptor for the following RPC calls:

  • GetSchema: return the Arrow schema of the query. Fields on this schema may contain the following metadata:
    • ARROW:FLIGHT:SQL:CATALOG_NAME - Tableโs catalog name
    • ARROW:FLIGHT:SQL:DB_SCHEMA_NAME - Database schema name
    • ARROW:FLIGHT:SQL:TABLE_NAME - Table name
    • ARROW:FLIGHT:SQL:TYPE_NAME - The data source-specific name for the data type of the column.
    • ARROW:FLIGHT:SQL:PRECISION - Column precision/size
    • ARROW:FLIGHT:SQL:SCALE - Column scale/decimal digits if applicable
    • ARROW:FLIGHT:SQL:IS_AUTO_INCREMENT - โ1โ indicates if the column is auto incremented, โ0โ otherwise.
    • ARROW:FLIGHT:SQL:IS_CASE_SENSITIVE - โ1โ indicates if the column is case-sensitive, โ0โ otherwise.
    • ARROW:FLIGHT:SQL:IS_READ_ONLY - โ1โ indicates if the column is read only, โ0โ otherwise.
