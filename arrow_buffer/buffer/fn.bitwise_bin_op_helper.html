where
    F: FnMut(u64, u64) -> u64,
Expand description

Apply a bitwise operation op to two inputs and return the result as a Buffer.

The inputs are treated as bitmaps, meaning that offsets and length are specified in number of bits.

NOTE: The operation op is applied to chunks of 64 bits (u64) and any bits outside the offsets and len are set to zero out before calling op.
